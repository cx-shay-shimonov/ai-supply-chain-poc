Initial commit: AI Supply Chain POC - Semantic Search & Static Analysis Tools

This POC demonstrates tools for detecting AI API and model usage in codebases using:
- Semantic search (sem) for natural language code queries
- Static analysis (semgrep) for pattern-based detection

## Features

### Core Scripts
- ai-usage-audit.sh: Main audit script combining semantic + semgrep analysis
- sem-audit.sh: Semantic-only audit (faster for AI model detection)
- sem-query.py: Non-interactive wrapper for automation and JSON output
- ai_asset_extractor.py: DRY module for extracting AI models/providers
- test.sh: Quick test script for semantic search validation

### Configuration
- my-detect-openai.yaml: Custom Semgrep rules for OpenAI/AI API detection
- shadow-ai-extended.yaml: Extended Semgrep ruleset for comprehensive AI detection
- .gitignore: Excludes venv/, output/, sample projects, and sensitive files

### Documentation
- README.md: Setup instructions and quick start guide
- SEM-EXAMPLES.md: Comprehensive semantic search examples (1,630 lines)
- SEMGREP-EXAMPLES.md: Static analysis examples and output formats (689 lines)
- FILE-STRUCTURE.md: Complete directory structure and file descriptions
- projects-samples/README.md: Sample project setup instructions

### Demo & Setup
- setup-and-demo.sh: Interactive setup with guided demos
- commands-examples.sh: 10+ working command examples
- test-samples.sh: Validation script for setup verification
